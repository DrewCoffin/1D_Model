Io Torus Widget
Graphical User Interface for a cubic centimeter model of the Io Plasma Torus

USER GUIDE - November 2012
Written by Adam Shinn

1   System Requirements

 - CHIANTI Atomic Database
 - IDL 8.1 or Later

1.1  CHIANTI Atomic Database

This widget requires the Chianti atomic database in order to produce simulated spectra. Information and download instructions will be covered in this guide, but can also be found here: http://www.chiantidatabase.org/

1.2  IDL 8.1 or later

IDL 8.1 and later is required due to the use of IDL's objected oriented plotting capabilities (IDL's plot function).

As the widget has been written and tested with UNIX-based operating systems, this guide will assume you are using a UNIX-based operating system.

2   How to get the widget up and running

2.1  Widget Package Contents

The widget package, toruswidget.tar.gz can be unpacked with your own preferred method, or the following method: 

 unix> gunzip toruswidget.tar.gz
 unix> tar xvf toruswidget.tar

Once unpacked, the toruswidget.tar.gz file contains a directory with the following contents: 

 - wtorus.pro           (handling procedure)
 - wtorus_idl81.pro     (main widget procedures)
 - torusmodel-data*.sav (data from model)
 - chianti/             (directory structure)

2.2  Downloading and setting up CHIANTI

Note: if you already have the Chianti atomic database on your system, please see the next section.

The Chianti atomic database can be downloaded as two compressed files from the CHIANTI website. Direct links to download each file are given here: 

 CHIANTI_7.1_pro_standalone.tar.gz  (IDL software)
 CHIANTI_7.1_data.tar.gz            (Chianti database)

Before unpacking the compressed files, place them in their appropriate directories, labeled "idl" and "dbase", within the Io Torus Widget directory structure as shown: 

 /.../toruswidget/chianti/idl/      (IDL software)
 /.../toruswidget/chianti/dbase/    (Chianti database)

Within the each directory, unpack both compressed files using the following method, or your own preferred method: 

 unix> gunzip [file_name].tar.gz
 unix> tar xvf [file_name].tar

Once each file is unpacked, the Chianti atmoic database should be properly installed on your system. For more information on downloading and setting up Chianti as a stand-alone package, visit http://www.chiantidatabase.org/download/.

2.3  Running the Widget

Once the Chianti atomic database is properly downloaded and unpacked within the "toruswidget/chianti/..." directory, then the widget is properly set up. If you already have the Chianti atomic database on your system, then open the handling procedure, wtorus.pro, and edit the variable "chiantipath" to equal to the appropriate path of the "chianti" directory on your system.

IMPORTANT: The handling procedure, wtorus.pro, starts up chianti in IDL as well as compiling and running the widget. Compiling the main procedure, wtorus_idl81.pro, is unnecessary and may cause errors.

To run the widget, simply run the handling procedure in the command line: 

 IDL> wtorus

If everything is set up properly, you will see dialogue of many procedures compiling, as well as a "Welcome to Chianti" dialogue that displays the values of assigned system variables: !xuvtop, !ioneq_file, and !abund_file.

2.4  Usage Tips

 - The input value slider bars are easily navigated when you use the arrow keys.
 - Scroll between sliders with [tab] and backwards with [shift + tab].
